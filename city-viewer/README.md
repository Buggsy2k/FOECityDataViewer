# FOE City Data Viewer

A React + Vite + TypeScript single-page app that visualizes Forge of Empires city data exports. Drag-and-drop a `citydata.json` file to explore production summaries, building details, a city grid map, and Great Buildings tracking.

## Prerequisites

- [Node.js](https://nodejs.org/) 18 or later

## Getting Started

```bash
cd city-viewer
npm install
npm run dev
```

The dev server starts at **http://localhost:5173**. Open it in your browser, then drag-and-drop a `citydata.json` file onto the page to load your city data.

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the Vite dev server with hot module replacement |
| `npm run build` | Type-check with TypeScript and build for production |
| `npm run preview` | Serve the production build locally for testing |
